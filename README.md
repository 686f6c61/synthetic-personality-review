# Synthetic Personality Review

Bitácora en actualización continua sobre «personalidad sintética» en modelos de lenguaje (LLMs).

- Sitio: https://syntheticpersonality.com
- Repositorio: https://github.com/686f6c61/synthetic-personality-review

## Qué aporta

Este proyecto reúne evidencia académica para entender cómo se evalúan, inducen y aplican rasgos de personalidad en LLMs.

El valor principal es práctico:

- concentrar literatura relevante en un solo lugar,
- separar estudios útiles de ruido o duplicados,
- facilitar comparación entre enfoques, años y fuentes,
- mantener una base de trabajo clara para investigación y producto.
